한 줄 비유

Claude Code는 천재 개발자보다 장비 지원이 빵빵한 환경에 가깝습니다.

중요한 건 모델 자체가 아니라, 배선·포트·검증 루프가 살아 있는 세팅입니다.

Model = CPU

중앙에서 추론하지만 전체를 보장하진 않는다.

Plans = 회로도

연결과 순서를 먼저 정리한다.

Subagents = 모듈 보드

역할별로 다른 처리 회로가 붙는다.

MCP / Tools = 포트

외부 세계와 닿는 연결 지점이다.

Playwright = 테스트 장비

실행 환경을 바로 확인하는 계측기다.

Review loop = 진단 모니터

이상 징후를 다시 구조 문제로 연결한다.
